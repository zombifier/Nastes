Group: Nastes

To Run Player Application
	Run PlayerMain.java in main package
	In the level selector menu, button "level 1" is unlock
To Run Builder Application
	Run BuilderMain.java in main package.
	Click on the piece in Scrollpane to add it to the bullpen(Piece that have dark color is piece on the bullpen)
	When press to drag the piece from the bullpen, the piece will not be able to drag back to the bullpen so you need to add the piece again
	Piece being selected is the red piece
To rotate a piece, use 'e' or 'r' key
To flip a piece, use 'q' or 'w' key
